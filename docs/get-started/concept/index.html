Concept

PyScript-React builds to different module types. By default it uses UMD.

Folder structure

/root
  - /components
    - /example-component
        - index.ts
        - example-component.story.tsx
        - example-component.test.tsx
        - example-component.tsx
        - example-component.types.ts
  - /hooks
    - /example-hook
      - index.ts
      - example-hook.ts
      - example-hook.types.ts
  - /stores
    - /example-store
      - index.ts
      - example-store.ts
      - example-store.types.ts
  - /types
    - /example-type
      - example-type.ts
  - index.ts # library entrypoint
